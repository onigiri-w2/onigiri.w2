## 基本のおにぎりの握り方

おにぎりの基本は「手早く」「優しく」です。以下の手順で作ってみましょう。

```plaintext
1. 手をきれいに洗い、少量の塩で手を整える
2. ご飯を適量（80〜100g程度）手のひらにのせる
3. 軽く握り、中央に窪みをつくって具材を入れる
4. 優しく全体を握り、三角形や俵型に整える
5. 海苔を巻く場合は、最後に巻きつける
```

> 🔑 **ポイント**：ご飯は熱すぎず冷たすぎない温度がベスト。強く握りすぎると固くなるので注意しましょう。

## 変わり種おにぎりアイデア

普段と違ったおにぎりを楽しみたい方には、以下のアイデアがおすすめです：

- **混ぜごはん系**
  - 炊き込みご飯おにぎり
  - ひじきご飯おにぎり
  - 五目ご飯おにぎり

- **トッピング系**
  - 天かすをまぶしたおにぎり
  - 焼きおにぎり（醤油・味噌）
  - イクラのせおにぎり

- **包み系**
  - 油揚げで包んだおにぎり（いなり風）
  - 卵焼きで包んだおむすび

## おにぎりの保存方法

おにぎりを美味しく保存するコツは以下の通りです：

1. **冷めてから包む**
   ```
   おにぎりが温かいうちにラップで包むと、水分が出て柔らかくなります。
   必ず冷ましてから包みましょう。
   ```

2. **海苔は食べる直前に巻く**
   ```
   海苔は湿気ると柔らかくなるので、食べる直前に巻くのがベスト。
   別に持ち運ぶか、海苔分離型のラップを使いましょう。
   ```

3. **冷凍保存のコツ**

   冷凍保存する場合は、以下の手順がおすすめです：

   ```bash
   # おにぎりの冷凍手順
   $ おにぎりを作る
   $ しっかり冷ます
   $ 一つずつラップでしっかり包む
   $ ジップロックに入れて空気を抜く
   $ 冷凍庫で保存（2週間程度が目安）
   
   # 解凍方法
   $ 自然解凍：ラップを外さずに30分程度置く
   $ 電子レンジ：ラップを緩めて、500Wで1分程度加熱
   $ トースター：解凍後、軽く焼くとおいしい
   ```
